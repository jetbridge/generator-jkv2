# JKv2 Yeoman generator
Opinionated generator for mono-repo web apps.

Features:
- No framework. Just functions and API Gateway
- Postgres & TypeORM

## How to use?
### Pre-requisites
#### npm 7
`npm i -g npm@7`
##### Yeoman
`npm i -g yo`
##### JKv2 generator
`npm i -g generator-jkv2`

### Scaffold the app
`yo jkv2`

### Add a new TypeORM DB model
`yo jkv2:model`
